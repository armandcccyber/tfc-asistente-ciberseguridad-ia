# TFC - Asistente de Ciberseguridad y Compliance basado en IA y RAG

**Alumno:** Armand Codina
**ID Scola:** EX0092
**Programa:** #Impact - Fundación GoodJob

## Descripción

Este proyecto consiste en el desarrollo de un asistente de ciberseguridad y compliance basado en Inteligencia Artificial y técnicas Retrieval-Augmented Generation (RAG).

El asistente ha sido implementado mediante n8n y OpenAI GPT, permitiendo consultar documentación normativa, responder preguntas de auditoría, detectar incumplimientos y generar propuestas de mejora basadas en documentación oficial previamente indexada.

## Tecnologías utilizadas

* n8n Cloud
* OpenAI GPT
* Embeddings OpenAI
* Vector Store
* Retrieval-Augmented Generation (RAG)
* Google Drive

## Consideraciones sobre la información utilizada

Este repositorio se publica exclusivamente con fines académicos como evidencia del Trabajo Fin de Curso del Programa #Impact.

La organización utilizada durante las pruebas (MidTech) es una entidad ficticia creada para la validación funcional del asistente y no contiene información real de empresas, clientes o personas físicas.

En un entorno profesional, la documentación normativa, evidencias de auditoría y datos de cumplimiento utilizados para entrenar o alimentar sistemas de inteligencia artificial estarían sujetos a políticas de confidencialidad y a la normativa aplicable en materia de protección de datos, por lo que no serían publicados en repositorios públicos sin las medidas de seguridad y autorizaciones correspondientes.
